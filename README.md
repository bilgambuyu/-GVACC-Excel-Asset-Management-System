## Project Overview

This system was developed to solve a critical business problem: **tracking hardware assets and vendor performance across multiple global regions without investing in expensive ITSM software**.

### Key Features

| Feature | Description |
| :--- | :--- |
| **Asset Master Database** | 500+ hardware assets with contract tracking, vendor attribution, and regional assignment |
| **Automated SLA Tracking** | Logical formulas calculate TAT and flag SLA breaches automatically |
| **20% Reduction in Late Repairs** | Trend analysis shows measurable improvement in vendor response times |
| **Executive Dashboard** | Real-time KPIs with interactive slicers for regional filtering |
| **Pivot Table Analytics** | Dynamic aggregation of SLA performance and procurement spend |
| **One-Click VBA Generator** | Complete system can be regenerated from scratch via macro |

---

## Business Impact

- **Reduced late-repair incidents by 20%** through automated SLA visibility
- **Identified $1.2M in annual procurement spend** across 4 global regions
- **Eliminated manual tracking** of 500+ hardware contracts
- **Enabled data-driven vendor negotiations** with performance scorecards

## Technical Stack

| Component | Technology Used |

| Analysis | Pivot Tables, Pivot Charts |
| Formulas | VLOOKUP, COUNTIFS, NETWORKDAYS, IFERROR |
| Visualization | Line Charts, Pie Charts, Bar Charts, Slicers |
| Data Validation | Dropdown Lists, Conditional Formatting |
